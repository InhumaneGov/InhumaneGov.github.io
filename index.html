Basic Usage

Just include the script in your HTML. That's it! 

<script src="https://inhumanegov.com/acknowledge-banner.js"></script>

Configuration Options

Attribute Default Description
data-theme light Color theme: 'light' or 'dark'
data-position bottom Banner position: 'top' or 'bottom'
data-expiration-days 365 How long to remember consent (in days)
data-message We use cookies to enhance your experience... Custom message text
data-accept-text I Understand Accept button text
data-learn-more-text Learn More Learn more button text
data-learn-more-url /privacy URL for learn more page
